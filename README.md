## ğŸ”§ Customization

### Update Contact Information
Edit the components:
- `Contact.jsx` - Phone, email, address
- `Footer.jsx` - Footer links
- `Hero.jsx` - Hero CTA links

### Update Booking Links
Replace `https://booking-link.com` in:
- `Booking.jsx`
- `Navbar.jsx`
- Service/Barber cards

### Update Images
Replace placeholder Unsplash URLs with your images in:
- `Hero.jsx` - Hero slider images
- `Services.jsx` - Service images
- `Barbers.jsx` - Barber photos
- `Gallery.jsx` - Gallery images
- `BeforeAfter.jsx` - Before/After images

### Update Colors
Edit CSS variables in `src/index.css`:
```css
:root {
    --primary-color: #1a1a1a;
    --accent-color: #d4af37;
    /* ... */
}
```
